## 📼 录像带式学习仓库
> A commit-driven programming learning repository.

这个仓库用于系统化学习与实践编程概念。

每一个知识点对应一个独立的 demo，
每一个 demo 对应一次独立的 commit。

通过 Git 历史回溯，可以恢复到当时的学习状态。

## 📌 设计理念

- 一个概念 = 一个 Demo
- 一个 Demo = 一个 Commit
- 当前工作区只保留“最新实验”
- 通过 commit hash 回顾历史知识点
- Markdown 负责索引与结构
- Git 负责时间轴与版本状态

这个仓库不是代码集合，而是：

> 一条可回溯的学习轨迹。

## 🎯 目标

* 构建系统化知识结构
* 强化概念的最小可运行实践
* 用 Git 记录学习轨迹
* 可长期扩展至多语言、多方向
